    Achúntale Web

    Achúntale Web surge como una necesidad de esta empresa para poder mostrar su equipo, su producto y tener una interfaz sencilla para poder realizar la compra de este.

    Este proyecto fue realizado con datos estáticos, y con las herramientas básicas del desarrollo web: HTML5, CSS3 y JavaScript.
