    Ayudevs

    Ayudevs es un proyecto realizado en conjunto a Luis Córdova y Javier Bagatoli, ambos integrantes del Programa de Profesionales Jóvenes. El objetivo de esta aplicación web recae en compartir conocimiento de las diversas herramientas que utiliza un desarrollador e ir seccionándolas dependiendo de su fin, por ejemplo, frontend, backend, entre otros.

    Este proyecto está siendo realizado con las utilidades básicas para el desarrollo web: HTML, CSS y JavaScript
